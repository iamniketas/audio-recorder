# Как другу установить приложение из GitHub Releases

## Что отправить другу
- Ссылку на страницу релизов:
  `https://github.com/iamniketas/audio-recorder/releases`

## Что ему нужно скачать
1. Открыть самый верхний релиз (Latest).
2. В блоке **Assets** скачать файл вида `*-Setup.exe`.
3. Не скачивать `*.nupkg`, `RELEASES`, `Source code`.

## Что нажать после скачивания
1. Запустить `*-Setup.exe`.
2. Подтвердить запуск (если Windows спросит).
3. После установки открыть `AudioRecorder` через меню Пуск.

## Как обновляться дальше
1. В приложении открыть левую панель "Настройки".
2. В блоке "Обновления" нажать `Проверить обновления`.
3. Дождаться загрузки.
4. Нажать `Установить обновление` (приложение перезапустится).

## Первая настройка Whisper
1. Если движок Whisper отсутствует, приложение предложит скачать `faster-whisper-xxl`.
2. После установки движка автоматически начнется загрузка модели `large-v2`.
3. Весь прогресс показывается прямо в секции "Транскрипция".
