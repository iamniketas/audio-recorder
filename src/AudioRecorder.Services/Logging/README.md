# Логирование в AudioRecorder

Этот каталог содержит компоненты логирования для приложения AudioRecorder.

## AppLogger.cs

Класс AppLogger предоставляет централизованную систему логирования для всего приложения.
Он использует NLog для записи логов в файл и консоль.

### Основные особенности:
- Логирование различных уровней: Trace, Debug, Info, Warn, Error, Fatal
- Автоматическое создание директорий для логов
- Добавлена проверка на создание лог-файла при инициализации
- Поддержка форматирования сообщений

### Использование:
```csharp
AppLogger.LogInfo("Информационное сообщение");
AppLogger.LogError("Ошибка: {0}", ex.Message);
```

### Формат логов:
- Дата и время
- Уровень логирования
- Имя метода
- Сообщение

### Конфигурация:
Файл конфигурации NLog.config определяет:
- Путь к лог-файлам
- Уровень логирования
- Формат записи
- Автоматическое создание директорий
